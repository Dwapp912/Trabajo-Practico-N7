# Trabajo-Practico-N7
## Contenido

El trabajo se divide en 4 ejercicios (Katas), todos contenidos en el paquete `trabajo.practicon7`.

* **Kata 1 (Herencia Básica):** `Vehiculo.java` y `Auto.java`. Demuestra la sobrescritura de métodos.
* **Kata 2 (Abstractos e Instanceof):** `Figura.java`, `Circulo.java`, `Rectangulo.java`. Demuestra el polimorfismo utilizando el patrón `instanceof` en la clase base para el cálculo de áreas.
* **Kata 3 (Sueldos y Clasificación):** `EmpleadoBase.java`, `EmpleadoPlanta.java`, `EmpleadoTemporal.java`. Demuestra el uso de `instanceof` para asignar comportamiento (sueldo) según el tipo de objeto.
* **Kata 4 (Comportamiento Sobrescrito):** `Animal.java`, `Perro.java`, `Gato.java`. Demuestra la sobrescritura de métodos (`@Override hacerSonido()`) para distintos tipos de objetos.

## Ejecución

Cada Kata debe ser ejecutada mediante su clase principal: `Main1.java` a `Main4.java`.
